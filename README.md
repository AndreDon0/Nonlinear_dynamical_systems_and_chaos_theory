### Домашняя работа №1 — Анализ устойчивости и бифуркация Хопфа

**Файл:** [`Homework_1.ipynb`](Homework_1.ipynb)

**Подробное условие:** [`1dz.pdf`](files/1dz.pdf)

**Вариант:** 7

**Система дифференциальных уравнений:**

$$\dot{x} = x(1-x) - \frac{axy}{1+bx}$$

$$\dot{y} = \frac{cxy}{1+bx} - y$$

**Цели работы:**

- Провести линейный анализ устойчивости стационарных точек для двумерной системы обыкновенных дифференциальных нелинейных уравнений.
- Определить значения управляющих параметров, при которых происходит бифуркация Хопфа, построить фазовые портреты до и после бифуркации.
- Найти мультипликаторы Флоке для численного анализа устойчивости предельного цикла.

**Методы и инструменты:**

- Аналитическое нахождение стационарных точек с помощью `SymPy`
- Вычисление матрицы Якоби и её собственных значений
- Анализ условий бифуркации Хопфа (след и определитель матрицы Якоби)
- Численное интегрирование системы ОДУ (`scipy.integrate.solve_ivp`)
- Построение фазовых портретов и временных рядов (`matplotlib`)
- Вычисление матрицы монодромии и мультипликаторов Флоке методом Эйлера
